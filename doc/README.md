# Documentation

This folder contains short feature/design docs for Project Etho.

## Purpose

Use these docs to capture behavior decisions, data contracts, and workflow notes that support the web app and pipeline.

## Naming

Docs use an ordered prefix format:

- `0000-<topic>.md`
- `0001-<topic>.md`
- `0002-<topic>.md`

Create the next number in sequence for new docs.
